1. Відкриваєте 3 або більше терміналів
2. На першому який обираєте сервером пишете "python server.py" або "python3 server.py" якщо у вас ця версія. Перед цим, звичайно, ви мали би перебувати в терміналі в тій папці, де знаходиться цей файл.
?
Також, якщо ви запустите сервер певторно, треба буде змінити номер сервера
?
3. В іншому терміналі, перейшовши попередньо в папку де лежить відповідний код, запустіть "python client.py" або "python3 client.py", знову ж таки, залежно від версії на вашому ноуті/ПК.
4. Введіть ім'я
5. Напишіть повідомлення
6. Для перевірки сервера має існувати мінімум 2 клієнти для спілкування, тому повторіть кроки з 3 по 5 включно для створення нового користувача. Повторюйте ці кроки щоразу, як захочете додати співрозмовника.
